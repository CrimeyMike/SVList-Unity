版本：V1.0.0 | 文档状态：正式版 | 作者：[你的名字] --- 第1章 引言与范围 1.1 项目背景 在Unity游戏开发中，UGUI原生ScrollRect配合Content Size Fitter在渲染大量物品（如背包、排行榜、聊天记录）时，会因生成全部GameObject导致内存暴涨和首帧卡顿。现有开源方案（如LoopScrollRect）存在动态高度支持差、API设计老旧、不支持多模板（Prefab）混合等痛点。 1.2 项目目标 开发一套开箱即用、高扩展性、支持百万级数据量的通用虚拟列表框架。核心指标： · 支持数据量：≥ 1,000,000 条。 · 运行帧率：快速滑动时保持 60fps。 · GC Allocation：滑动过程中 0KB 堆内存分配。 · 配置方式：完全通过Inspector面板可视化配置。 --- 第2章 软件工程需求定义 2.1 功能性需求（FR） · FR-01：支持水平和垂直两种滚动方向。 · FR-02：支持固定高度与动态高度（根据内容自适应）混合布局。 · FR-03：支持多模板（Multi-Template），即同一个列表内可渲染多种不同尺寸的Prefab。 · FR-04：内置对象池（Object Pool），严格管控GameObject的实例化与销毁。 · FR-05：提供运行时调试面板，实时显示当前可视元素数量与内存占用。 2.2 非功能性需求（NFR） · NFR-01（性能）：单帧实例化GameObject数量不超过 Max Instantiate Per Frame（默认5个），超标时分帧加载。 · NFR-02（鲁棒性）：极端快速滑动时，不出现“白屏闪烁”或“索引越界”异常。 · NFR-03（兼容性）：兼容 Unity 2021.3 LTS 及以上版本，支持 Render Pipeline 转换。 --- 第3章 总体架构设计（4+1视图） 3.1 模块分解图（逻辑视图） 层级 模块名称 职责 应用层 SampleScene 挂载具体业务数据（如道具ID、文本内容）。 桥接层 ISVDataSource<T> 数据源接口，业务层需实现GetItemCount()与GetItemData(index)。 核心层 SVListCore 无MonoBehaviour的纯逻辑类。负责可见范围计算、滚动偏移量换算、索引映射。 表现层 SVItemRenderer 挂载在单个Item上，负责SetData(T data)刷新UI图文，并上报自身Height。 管理层 SVObjectPool 负责Prefab的实例化、回收、扩容。支持预热（Pre-warm）。 配置层 SVConfig 被Inspector序列化，包含边距、间距、预估尺寸、预加载阈值等。 3.2 核心数据结构定义

--- 第4章 详细设计（关键算法与类设计） 4.1 核心算法：二分查找与锚点换算 · 输入：ScrollRect.normalizedPosition 转换为 float scrollY。 · 处理： 1. 通过 二分法（Binary Search） 在 _cachedItemHeights 中查找第一个使 累积高度 > scrollY 的索引，作为 FirstVisibleIndex。 2. 计算 FirstVisibleIndex 之前的总高度 TopHeight。 3. 将 Content 的 anchoredPosition 设为 new Vector2(0, TopHeight)（垂直模式）。 · 关键代码接口：

4.2 对象池回收机制（双缓冲策略） · 设计原则：不在滑动时立即销毁移出视图的Item，而是放入待回收队列（Pending Recycle Queue）。 · 触发条件：当待回收队列长度 > 对象池当前活跃对象数 * 0.3 时，批量执行Recycle()。 · 原因：防止频繁调用SetParent(null)触发UGUI的脏标记重建。 4.3 Inspector 面板配置参数详解 参数分组 参数名 类型 默认值 说明 Base Viewport Padding Vector4 (5,5,5,5) 列表内容与视口边界的四边距。 Base Spacing Vector2 (10,10) 元素之间的水平和垂直间隙。 Estimate Default Cell Size float 100 动态高度计算前的预估基准值（影响初始占位）。 Performance Preload Threshold float 0.5 视口外预加载的数据量倍数（0.5即提前加载半屏）。 Performance Max Pool Size int 50 对象池最大驻留数量，防止内存无限膨胀。 Debug Show Gizmos bool false 在Scene视图中绘制视口边界和Content轮廓。 --- 第5章 接口API设计（面向业务开发者） 5.1 初始化接口
